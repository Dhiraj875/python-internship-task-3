# python-internship-task-3
Python grade calculator using conditional statements
This project is part of the Python Developer Internship.

## Task Description
A grade calculator program that:
- Takes marks as input
- Uses if-elif-else conditions
- Uses logical operators
- Handles invalid input
- Displays grades and performance messages

## Technologies Used
- Python
- VS Code

## Output
The program displays the grade based on marks entered by the user.
